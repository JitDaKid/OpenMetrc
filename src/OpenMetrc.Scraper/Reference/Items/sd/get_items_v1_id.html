Items

Server: https://api-sd.metrc.com/

GET /items/v1/{id}

Permissions Required

Manage Items

Parameters

licenseNumber
optional		If specified, the Item will be validated against the specified License Number. If not specified, the Item will be validated against all of the User's current Facilities. Please note that if the Item is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /items/v1/1?licenseNumber=123-ABC
