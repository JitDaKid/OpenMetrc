Items

Server: https://api-ak.metrc.com/

POST /items/v2/file

Uploads one or more image or PDF files for products, labels, packaging, or documents at the specified Facility.

Permissions Required

Manage Items

Parameters

licenseNumber

Example Request

POST /items/v2/file?licenseNumber=123-ABC[
  {
    "FileName": "Image.png",
    "EncodedImageBase64": "File encoded in Base64=="
  }
]

Example Response

{
  "Ids": [
    1
  ],
  "Warnings": null
}
