Items

Server: https://api-md.metrc.com/

GET /items/v2/file/{id}

Retrieves a file by its Id for the specified Facility.

Permissions Required

Manage Items

Parameters

licenseNumber
optional		The License Number of the Facility for which to return the file.

Example Request

GET /items/v2/file/1?licenseNumber=123-ABC

Example Response

{
  "FileContents": "iVBORw0KGgoAAAANSUhEUg...[truncated Base64 string]",
  "ContentType": "application/octet-stream",
  "FileDownloadName": "gummies.png"
}
