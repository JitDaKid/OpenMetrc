Items

Server: https://api-ri.metrc.com/

GET /items/v1/categories

Permissions Required

None

Parameters

licenseNumber
optional		If specified, the Categories will be retrived for the specified License Number. If not specified, the all Item Categories will be returned.

Example Request

GET /items/v1/categories?licenseNumber=123-ABC
