Items

Server: https://api-mt.metrc.com/

GET /items/v1/brands

Permissions Required

Manage Items

Parameters

licenseNumber
The license number of the facility for which to return the list of active item brands.

Example Request

GET /items/v1/brands?licenseNumber=123-ABC

Example Response

[
  {
    "Name": "Skunk Nasty",
    "Status": "Paid"
  },
  {
    "Name": "Darth Buds",
    "Status": "NotPaid"
  }
]
