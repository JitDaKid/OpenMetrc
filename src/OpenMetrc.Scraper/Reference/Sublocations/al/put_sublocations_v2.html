Sublocations

Server: https://api-al.metrc.com/

PUT /sublocations/v2/

Updates existing sublocation records for a specified Facility.

Permissions Required

Manage Locations

Parameters

licenseNumber
The License Number of the Facility for which to update the list of sublocations.

Example Request

PUT /sublocations/v2/?licenseNumber=123-ABC[
  {
    "Id": 1,
    "Name": "Room 1A Sublocation"
  },
  {
    "Id": 2,
    "Name": "Table 2B Sublocation"
  }
]

Example Response

No response
