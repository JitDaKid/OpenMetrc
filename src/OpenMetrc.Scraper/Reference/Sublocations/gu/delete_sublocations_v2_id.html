Sublocations

Server: https://api-gu.metrc.com/

DELETE /sublocations/v2/{id}

Archives an existing Sublocation record for a Facility.

Permissions Required

Manage Locations

Parameters

licenseNumber
The License Number of the Facility for which to delete the list of sublocations.

Example Request

DELETE /sublocations/v2/5?licenseNumber=123-ABC

Example Response

No response
