Units Of Measure

Server: https://api-wv.metrc.com/

GET /unitsofmeasure/v2/inactive

Retrieves all inactive units of measure.

Permissions Required

None

Parameters

No parameters

Example Request

GET /unitsofmeasure/v2/inactive

Example Response

[
  {
    "QuantityType": "WeightBased",
    "Name": "Tonns",
    "Abbreviation": "t"
  }
]
