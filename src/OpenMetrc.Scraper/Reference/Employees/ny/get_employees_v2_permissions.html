Employees

Server: https://api-ny.metrc.com/

GET /employees/v2/permissions

Retrieves the permissions of a specified Employee, identified by their Employee License Number, for a given Facility.

Permissions Required

Manage Employees

Parameters

licenseNumber
The License Number of the Facility under which to get the Employee.
employeeLicenseNumber
The License Number of the Employee under which to get the Permission.

Example Request

GET /employees/v2/permissions?licenseNumber=123-ABC&employeeLicenseNumber=ELN-0000000000005

Example Response

[
  "view immature plants",
  "view packages",
  "view plants"
]
