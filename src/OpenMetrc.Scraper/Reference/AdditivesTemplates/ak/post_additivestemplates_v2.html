Additives Templates

Server: https://api-ak.metrc.com/

POST /additivestemplates/v2/

Creates new additive templates for a specified Facility.

Permissions Required

Manage Additives

Parameters

licenseNumber
The License Number of the Facility for which to record the list of additives.
