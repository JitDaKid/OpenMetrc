Patients

Server: https://api-ma.metrc.com/

GET /patients/v1/active

Permissions Required

Manage Patients

Parameters

licenseNumber
The license number of the facility for which to return the list of active patients.

Example Request

GET /patients/v1/active?licenseNumber=123-ABC
