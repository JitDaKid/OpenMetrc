Patients

Server: https://api-dc.metrc.com/

GET /patients/v1/{id}

Permissions Required

Manage Patients

Parameters

licenseNumber
optional		If specified, the Patient will be validated against the specified License Number. If not specified, the Patient will be validated against all of the User's current Facilities. Please note that if the Patient is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /patients/v1/1?licenseNumber=123-ABC
