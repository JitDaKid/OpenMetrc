Plant Batches

Server: https://api-ca.metrc.com/

POST /plantbatches/v2/plantings

Creates new plantings for a Facility by generating plant batches based on provided planting details.

Permissions Required

View Immature Plants
Manage Immature Plants Inventory

Parameters

licenseNumber
The License Number of the Facility for which to create the plantings.
