Harvests

Server: https://api-oh.metrc.com/

GET /harvests/v2/{id}

Permissions Required

View Harvests

Parameters

licenseNumber
optional		If specified, the Harvest will be validated against the specified License Number. If not specified, the Harvest will be validated against all of the User's current Facilities. Please note that if the Harvest is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /harvests/v2/1?licenseNumber=123-ABC
