Harvests

Server: https://api-sd.metrc.com/

PUT /harvests/v2/location

Updates the Location of Harvest for a specified Facility.

Permissions Required

View Harvests
Manage Harvests

Parameters

licenseNumber
The License Number of the Facility for which to return the list of harvests locations.
