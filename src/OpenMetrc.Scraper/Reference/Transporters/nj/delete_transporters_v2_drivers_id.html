Transporters

Server: https://api-nj.metrc.com/

DELETE /transporters/v2/drivers/{id}

Archives a Driver record for a Facility. Please note: The {id} parameter above represents a Driver Id.

Permissions Required

Manage Transporters

Parameters

licenseNumber
The License Number of the Facility for which to archive a Driver.

Example Request

DELETE /transporters/v2/drivers/?licenseNumber=123-ABC

Example Response

No response
