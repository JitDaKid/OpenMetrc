Processing Jobs

Server: https://api-mi.metrc.com/

GET /processing/v2/jobtypes/attributes

Retrieves all processing job attributes available for a Facility.

Permissions Required

Manage Processing Job

Parameters

licenseNumber
The License Number of the Facility for which to return the list of active processing job type attributes.

Example Request

GET /processing/v2/jobtypes/attributes?licenseNumber=123-ABC
