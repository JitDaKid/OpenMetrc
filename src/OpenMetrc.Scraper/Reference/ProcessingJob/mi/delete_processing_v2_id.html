Processing Jobs

Server: https://api-mi.metrc.com/

DELETE /processing/v2/{id}

Archives a Processing Job at a Facility by marking it as inactive and removing it from active use.

Permissions Required

Manage Processing Job

Parameters

licenseNumber
The License Number of the Facility of the Processing Job.

Example Request

DELETE /processing/v2/7?licenseNumber=123-ABC

Example Response

No response
