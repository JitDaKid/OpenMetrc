Processing Jobs

Server: https://api-gu.metrc.com/

PUT /processing/v2/finish

Completes processing jobs at a Facility by recording final notes and waste measurements.

Permissions Required

Manage Processing Job

Parameters

licenseNumber
The License Number of the Facility to finish the Processing Job.

Example Request

PUT /processing/v2/finish?licenseNumber=123-ABC[
  {
    "Id": 1,
    "FinishDate": "2021-11-23T00:00:00Z",
    "FinishNote": "Infusing",
    "TotalCountWaste": null,
    "WasteCountUnitOfMeasureName": null,
    "TotalVolumeWaste": null,
    "WasteVolumeUnitOfMeasureName": null,
    "TotalWeightWaste": 10.0,
    "WasteWeightUnitOfMeasureName": "Ounces"
  }
]

Example Response

No response
