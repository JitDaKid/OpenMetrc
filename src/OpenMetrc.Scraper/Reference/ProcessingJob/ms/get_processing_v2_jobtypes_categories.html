Processing Jobs

Server: https://api-ms.metrc.com/

GET /processing/v2/jobtypes/categories

Retrieves all processing job categories available for a specified Facility.

Permissions Required

Manage Processing Job

Parameters

licenseNumber
The License Number of the Facility for which to return the list of processing job type categories.

Example Request

GET /processing/v2/jobtypes/categories?licenseNumber=123-ABC
