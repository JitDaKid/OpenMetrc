Processing Jobs

Server: https://api-va.metrc.com/

PUT /processing/v2/unfinish

Reopens previously completed processing jobs at a Facility to allow further updates or corrections.

Permissions Required

Manage Processing Job

Parameters

licenseNumber
The License Number of the Facility to unfinish the Processing Job.

Example Request

PUT /processing/v2/unfinish?licenseNumber=123-ABC[
  {
    "Id": 1
  }
]

Example Response

No response
