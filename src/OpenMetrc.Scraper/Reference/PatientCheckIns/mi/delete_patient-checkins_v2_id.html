Patient Check Ins

Server: https://api-mi.metrc.com/

DELETE /patient-checkins/v2/{id}

Archives a Patient Check-In, identified by its Id, for a specified Facility.

Permissions Required

ManagePatientsCheckIns

Parameters

licenseNumber
The License Number of the Facility for which to archive Patient check-in.

Example Request

DELETE /patient-checkins/v2/4?licenseNumber=123-ABC

Example Response

No response
