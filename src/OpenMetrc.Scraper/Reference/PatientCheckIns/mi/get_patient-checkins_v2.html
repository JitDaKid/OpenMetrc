Patient Check Ins

Server: https://api-mi.metrc.com/

GET /patient-checkins/v2/

Retrieves a list of patient check-ins for a specified Facility.

Permissions Required

ManagePatientsCheckIns

Parameters

licenseNumber
The License Number of the Facility for which to return the list of Patient check-ins.
checkinDateStart
optional		The last modified start timestamp
checkinDateEnd
optional		The last modified end timestamp

Example Request

GET /patient-checkins/v2/?licenseNumber=123-ABC&checkinDateStart=2018-01-17T06:30:00Z&checkinDateEnd=2018-01-17T17:30:00Z
