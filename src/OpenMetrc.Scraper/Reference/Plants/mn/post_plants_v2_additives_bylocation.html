Plants

Server: https://api-mn.metrc.com/

POST /plants/v2/additives/bylocation

Records additive usage for plants based on their location within a specified Facility.

Permissions Required

Manage Plants
Manage Plants Additives

Parameters

licenseNumber
The License Number of the Facility for which to record Plant additives.
