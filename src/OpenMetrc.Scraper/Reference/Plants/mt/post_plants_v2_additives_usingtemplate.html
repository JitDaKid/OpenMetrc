Plants

Server: https://api-mt.metrc.com/

POST /plants/v2/additives/usingtemplate

Records additive usage for plants using predefined additive templates at a specified Facility.

Permissions Required

Manage Plants Additives

Parameters

licenseNumber
The License Number of the Facility for which to record Plant additives.

Example Request
