Plants

Server: https://api-ri.metrc.com/

GET /plants/v1/{label}

Permissions Required

View Veg/Flower Plants

Parameters

licenseNumber
optional		If specified, the Plant will be validated against the specified License Number. If not specified, the Plant will be validated against all of the User's current Facilities. Please note that if the Plant is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /plants/v1/ABCDEF012345670000000024?licenseNumber=123-ABC
