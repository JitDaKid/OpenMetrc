Transfers

Server: https://api-ky.metrc.com/

GET /transfers/v2/{id}/deliveries

Retrieves a list of shipment deliveries for a given Transfer Id. Please note: The {id} parameter above represents a Transfer Id.

Permissions Required

Manage Transfers
View Transfers

Parameters

pageNumber
optional
The number of the data page from which to return data.
pageSize
optional
The number of records to return per page. Pagination is currently disabled by default. You can enable pagination on this query by specifying a value that does not exceed 20.

Example Request

GET /transfers/v2/1/deliveries?pageNumber=1&pageSize=10
