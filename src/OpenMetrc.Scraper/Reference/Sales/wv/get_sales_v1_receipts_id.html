Sales

Server: https://api-wv.metrc.com/

GET /sales/v1/receipts/{id}

Permissions Required

Sales

Parameters

licenseNumber
optional		If specified, the Sales Receipt will be validated against the specified License Number. If not specified, the Sales Receipt will be validated against all of the User's current Facilities. Please note that if the Sales Receipt is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /sales/v1/receipts/1?licenseNumber=123-ABC
